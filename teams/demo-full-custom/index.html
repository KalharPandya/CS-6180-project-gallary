NLP Sentiment Analysis BERT Interactive

Abstract

We fine-tuned BERT-base on a combined corpus of 240k Amazon, Yelp, and Twitter reviews for multi-domain sentiment classification. Our model achieves 94.1% accuracy on a held-out test set and generalizes well across domains. Below is a live demo running inference client-side via ONNX Runtime Web.

Approach

We fine-tuned bert-base-uncased for 3 epochs using AdamW (lr=2e-5, warmup=10%). Domain adaptation was applied via continued pre-training on in-domain unlabeled text before fine-tuning. The model is exported to ONNX and quantized for client-side inference.
